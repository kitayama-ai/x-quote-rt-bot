# 海外AIバズ引用RT自動化ツール — 操作マニュアル

> バージョン: v2.0 / 最終更新: 2026-02-22

---

## このツールで何ができるか

海外のAI系バズツイートを自動で見つけて、日本語コメントをAIが生成し、Xに引用RTとして投稿します。

```
① 海外バズツイートを自動収集
       ↓
② AIが日本語コメントを8パターンから自動生成
       ↓
③ あなたが承認 or 完全自動で投稿（選択可能）
```

**すべての操作はダッシュボード（ブラウザ）から行えます。ターミナルは不要です。**

---

## ダッシュボード

**URL: https://isai-11f7b.web.app**

- **X（Twitter）アカウントでログイン**します（Googleログインは廃止済み）

## 初期設定（初回のみ）

### ステップ 1: APIキーを登録する

**ダッシュボード → 「設定」タブ → APIキー設定**

以下のキーを登録してください：

| キー名 | 取得場所 | 用途 |
|---|---|---|
| **Gemini API Key** | [aistudio.google.com/apikey](https://aistudio.google.com/apikey) | AIコメント生成 + 日本語要約 |
| **X API Bearer Token** | [developer.x.com](https://developer.x.com/en/portal/dashboard) | ツイート収集 |
| **X API Key / Secret** | 同上 → Keys and Tokens | X投稿認証 |
| **X Access Token / Secret** | 同上 → Keys and Tokens | X投稿認証 |
| Discord Webhook URL（任意） | Discord チャンネル設定 → ウェブフック | 投稿通知 |

> キーはFirestoreに暗号化保存され、あなた自身しかアクセスできません。

### ステップ 2: 運用モードを選ぶ

**ダッシュボード → 「設定」タブ → 運用モード**

| モード | 動作 | おすすめ |
|---|---|---|
| 🛡️ **半自動（承認制）** | AIが生成 → あなたが承認 → 投稿 | 最初はこちらがおすすめ |
| 🚀 **完全自動** | AIが生成 → 自動承認 → 自動投稿 | 慣れたら切り替え |

## 日常の使い方

### 半自動モード（承認制）の場合

1. **キュー一覧タブ**を開く
2. 収集されたツイートが一覧で表示される
3. **行をクリック** → 詳細モーダルが開く
   - 元ツイート全文（英語）
   - **日本語要約**（Gemini APIで自動翻訳）
   - AI生成コメント全文（投稿される文面）
   - エンゲージメント（❤️いいね 🔁RT 💬リプ）
4. 内容を確認して **「承認」** or **「スキップ」** をクリック
5. 承認したツイートは自動投稿スケジュールで投稿される

### 完全自動モードの場合

基本的に何もする必要はありません。定期的にダッシュボードで投稿状況を確認してください。

## 手動操作（操作タブ）

通常は自動で動きますが、今すぐ実行したい場合はボタンから操作できます。

**ダッシュボード → 「操作」タブ**

| ボタン | 動作 |
|---|---|
| 📡 **今すぐ収集** | 海外バズツイートを即座に収集 |
| ✍️ **今すぐ生成** | 承認済みツイートにAIコメントを生成 |
| 📤 **今すぐ投稿** | 生成済みの引用RTをXに投稿 |
| 📊 **今すぐ更新** | ダッシュボードのデータを最新に更新 |

### ツイートURL手動追加

引用RTしたい特定のツイートがある場合：

1. **操作タブ** → 「ツイートURLを手動追加」
2. ツイートのURLを貼り付けて「追加」
3. 「今すぐ生成」→「今すぐ投稿」の順でボタンを押す

## AIコメントのカスタマイズ（選定設定タブ）

### 引用RTプロンプト設定

| 設定項目 | 説明 |
|---|---|
| **ペルソナ名** | AIが演じるキャラクター名 |
| **一人称** | 「僕」「私」「俺」等 |
| **ポジション** | 自分の立場（例: AI活用の第一人者） |
| **差別化要素** | 他と何が違うか |
| **口調・トーン** | コメントの雰囲気 |
| **見出しパターン** | 使うヘッドラインのスタイル |
| **NGワード** | 使ってはいけない単語 |
| **カスタム指示** | AIへの追加指示（自由記述） |

### テンプレート（8パターン）

AIが自動で最適なテンプレートを選びます。使わないパターンはOFFにできます。

| テンプレート | 見出し例 |
|---|---|
| 市場インパクト型 | 💥【衝撃】〇〇がXXXを発表 |
| 要点まとめ型 | 📌【要約】3つのポイント |
| 警告・問題提起型 | 🚨 これ、マジでやばい |
| 激震分析型 | ⚡【激震】業界地図が変わる |
| 衝撃速報型 | 🔥【速報】海外で〇〇が... |
| 独占入手型 | 💥【独占】最新情報を入手 |
| ダーク警告型 | 💀 これを知らないと... |
| 伝説・歴史型 | 💥【伝説】歴史的瞬間 |

> 毎回異なるパターンが選ばれるよう、3層のバリエーション制御（テンプレートローテーション、プロンプト注入、1日2回制限）が動作します。

### その他の設定

| 設定 | 説明 |
|---|---|
| **週間フォーカス** | 今週特に注目したいテーマ |
| **注目/回避トピック** | 収集ジャンルの優先度調整 |
| **ブースト/ブロック** | 特定の海外アカウントの優先or除外 |
| **収集条件** | 最大取得件数、最低いいね数、最大経過時間 |

## トラブルシューティング

### 投稿が止まっている

**確認場所:** GitHub Actions の実行履歴

赤い✗が出ているワークフローをクリックしてエラー内容を確認してください。

| エラー | 原因 | 対処 |
|---|---|---|
| 403 Forbidden | X APIスパム判定 | 30分待って再実行 |
| 402 Payment Required | X APIクレジット残高ゼロ | console.x.com → Billing でクレジット購入 |
| 401 Unauthorized | アクセストークン期限切れ | 設定タブでTokenを再登録 |
| ダッシュボードにデータが表示されない | dashboard_dataが未生成 | 操作タブ →「今すぐ更新」ボタン |

### 日本語要約が表示されない

「設定」タブで **Gemini API Key** が登録されているか確認してください。
キーが未登録だと、詳細モーダルの日本語要約は表示されません。

### ログインできない

- X Developer Portalのコールバック URL に `https://isai-11f7b.firebaseapp.com/__/auth/handler` が登録されているか確認
- X API の OAuth 設定が有効になっているか確認
