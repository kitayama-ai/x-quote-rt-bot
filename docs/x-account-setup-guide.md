# 🛡️ Xアカウント作成 — 凍結回避 完全ガイド

> 新規アカウントは作成直後が最もBANリスクが高い。
> **最初の10日間の行動が全てを決める。**

---

## ⚠️ なぜ新規アカウントはBANされやすいのか

X（旧Twitter）は新規アカウントを「ボット / スパムの可能性あり」として**強い監視対象**にしている。
以下のいずれかに該当するとほぼ即凍結：

- 作成直後に大量フォロー / いいね / 投稿
- プロフィール未設定のまま活動開始
- 過去にBANされたメアド / 電話番号 / IP の使い回し
- 自動ツールによる不自然な操作パターン

---

## 📋 アカウント作成前の準備

| #   | 準備項目                 | 詳細                                               |
| --- | ------------------------ | -------------------------------------------------- |
| 1   | **新しいメールアドレス** | Gmail OK。過去にBAN歴のあるメアドは絶対NG          |
| 2   | **電話番号**             | SMSが受信できる番号。認証に必要                    |
| 3   | **プロフィール素材**     | アイコン画像、ヘッダー画像、自己紹介文を事前に用意 |
| 4   | **通常のブラウザ環境**   | VPN不要。自宅WiFiかモバイル回線で作成              |
| 5   | **投稿ネタ5本**          | 最初の手動投稿用。AIではなく自分で書いたもの       |

> [!CAUTION]
> 過去BANされたアカウントと**同じIPアドレス・同じ端末**で作成すると、即座に関連BANされる可能性あり

## 🚀 作成直後にやること（Day 0 — 作成日）

### ✅ 必須（上から順番に）

```
1. メール認証を完了する
2. 電話番号を登録 & SMS認証を完了する
3. プロフィール画像を設定する（デフォルト卵アイコンのままは危険）
4. ヘッダー画像を設定する
5. 自己紹介文を書く（160文字しっかり埋める）
6. 場所を設定する（日本）
7. Webサイト欄は空欄のままでOK（初日にURL貼るとスパム判定）
```

### ✅ 初日の行動

```
8. 興味のあるジャンルのアカウントを 5〜10人 フォローする
   └ 公式アカウントや有名人が安全（NHK, 孫正義, etc.）
9. フィードを 10〜15分 スクロールして眺める
10. 気になった投稿に「いいね」を 3〜5個 つける
11. 1件だけリプライする（「勉強になります！」程度でOK）
```

### ❌ 初日にやってはいけないこと

```
× 投稿する（Day 0は投稿しない）
× 10人以上フォローする
× DMを送る
× 外部リンクを貼る
× ハッシュタグを大量に使う
× 自動化ツールを接続する
```

## 📅 最初の10日間スケジュール

### Day 1-2（閲覧 & 軽いエンゲージメント）

| やること             | 数量       | 目的                   |
| -------------------- | ---------- | ---------------------- |
| フィードを眺める     | 10-15分/日 | 利用時間の実績を作る   |
| いいね               | 5-10個/日  | アクティブユーザー認定 |
| リプライ             | 1-3件/日   | 会話の実績を作る       |
| フォロー             | 3-5人/日   | 徐々に増やす           |
| **投稿はまだしない** | 0          | —                      |

### Day 3-5（初投稿 & 交流開始）

| やること                 | 数量       | 目的                     |
| ------------------------ | ---------- | ------------------------ |
| **初投稿**（自己紹介系） | 1本/日     | アカウントの方向性を示す |
| いいね                   | 10-15個/日 | 少しずつ増やす           |
| リプライ                 | 3-5件/日   | 他の人との交流           |
| フォロー                 | 5-10人/日  | ターゲット層を少しずつ   |
| RT（引用なし）           | 1-2件/日   | 自然な拡散行動           |

> [!TIP]
> **初投稿のおすすめ内容**
>
> - 「AI×副業に興味があって始めました！同じ仲間いたら繋がりましょう💪」
> - 自分の経験・感想ベースの内容が安全
> - テンプレ感ゼロの自然な文章にする

### Day 6-10（通常運用への移行）

| やること | 数量       | 目的                 |
| -------- | ---------- | -------------------- |
| 投稿     | 1-2本/日   | 投稿頻度を上げる     |
| いいね   | 15-20個/日 | —                    |
| リプライ | 5-10件/日  | エンゲージメント     |
| フォロー | 5-10人/日  | ターゲット層を広げる |
| 引用RT   | 1-2件/日   | 自分の意見を添える   |

### Day 11以降（自動投稿への移行準備）

```
✅ ここまで生き残れば、アカウントの信頼スコアは一定レベルに達している

→ X Developer Portal でアプリ作成（APIキー取得）
→ x-auto-post-system を接続
→ manual_approval モードで自動生成 → 手動投稿から開始
→ 2週間問題なければ semi_auto モードに移行
```

## 🔢 X公式の行動制限（知っておくべき数字）

| 項目     | 上限             | 備考                            |
| -------- | ---------------- | ------------------------------- |
| 投稿     | 2,400件/日       | 30分ごとに分割制限あり          |
| DM       | 500件/日         | 新規は特に厳しく監視            |
| フォロー | 400人/日         | 新規は50人/日でも危険           |
| いいね   | 制限なし（公式） | ただし短時間に100件以上は怪しい |
| リプライ | 制限なし（公式） | 同じ文面の連投はスパム判定      |

> [!WARNING]
> 上記は「上限」であり「安全値」ではない。新規アカウントはこの **1/10 以下** で行動すること

## 🚫 絶対にやってはいけないこと（全期間共通）

### 即BAN級

- ❌ 同一内容のコピペ投稿を複数アカウントに投稿
- ❌ フォロー→フォロバ来たら即アンフォローを繰り返す
- ❌ 大量の一方的なDM送信
- ❌ エンゲージメントファーミング（相互いいねグループ等）
- ❌ 他人のなりすまし
- ❌ 暴力・ヘイト・違法コンテンツ

### 高リスク

- ⚠️ 短時間に100件以上のいいね
- ⚠️ プロフィール画像なしでの活動
- ⚠️ 投稿にURLを毎回含める
- ⚠️ 同じハッシュタグの多用
- ⚠️ アカウント名の頻繁な変更
- ⚠️ 認証されていない自動化ツールの使用

## 💡 自動投稿開始後の注意事項

1. **投稿間隔は最低4時間空ける**（1日3投稿: 7時, 12時, 21時）
2. **毎回異なる文面にする**（重複検出はシステムに実装済み）
3. **URLは投稿本文に入れない**（プロフのリンクに誘導）
4. **最初の2週間は manual_approval モードで手動確認**
5. **凍結されたら即座に自動投稿を停止** → 異議申し立て
