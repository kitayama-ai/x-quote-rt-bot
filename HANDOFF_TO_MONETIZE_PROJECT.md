## ✅ 決定事項

### 1. 3アカウント戦略

- **アカウント①**: AIエージェント・自動化（上級者向け）★まずここから
- **アカウント②**: AIツール紹介・レビュー（全レベル向け）
- **アカウント③**: LLM・テキスト生成AI入門（初心者向け）
- **段階展開**: ①を完成させてから②→③の順に横展開

### 2. 投稿設計

- **各アカウント1日3投稿**（朝・昼・夜）
- 投稿時間はランダムで±15分ずらす（凍結対策）
- 宣伝は全投稿の20%以下

### 3. 自動化レベル（段階的）

- **Phase 1（1-2週目）**: 生成は自動、投稿は手動承認
- **Phase 2（3-4週目）**: スコア8点以上は自動投稿
- **Phase 3（5週目〜）**: 安全チェック通過で全自動

### 4. 技術スタック

- Python + GitHub Actions + Gemini API + X API (Free tier) + Discord Webhook + Google Sheets
- 全部無料 or ほぼ無料

### 5. 通知先: Discord

- 会社（Slack）とは分離してDiscordに
- チャンネル構成: アカウント別 + メトリクス + 安全アラート

### 6. コスト

- アカウント①のみ: **月100〜300円**
- 3アカウント運用: **月300〜1,250円**

### 7. リスク管理

- 安全チェッカー（NGワード、スコアリング、重複検出、会社情報漏洩防止）
- アカウント凍結対策（固有メール・電話番号、暖機運転、投稿間隔4h以上）
- クロスプロモーション週2回まで

### 8. 既存資産の活用

- **x-monetize-project**: PDCA設計、プロンプト集、投稿テンプレ、競合分析
- **ai-tweet-collector**: GAS自動投稿、RSS→翻訳パイプライン
- **x_tweet_collector**: 高エンゲージツイート収集
- **x-viral-tweet-generator**: アルゴリズム準拠リライト、GitHub Actions雛形、予算管理
