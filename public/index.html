📖 このツールの使い方

X（Twitter）引用RT自動投稿ツールの操作ガイドです。

🔧 ステップ 0: 初期設定
1. 設定タブで各種APIキーを登録
   • Gemini API Key（コメント生成AI）
   • X API各種キー（Bearer Token, API Key/Secret, Access Token/Secret）
   • Discord Webhook URL（通知用・任意）
2. 運用モードを選択（半自動 or 完全自動）
📡 ステップ 1: バズツイート収集
海外AI系インフルエンサーのバズツイート（英語、500いいね以上、48時間以内）を自動収集。
• 操作タブの「今すぐ収集」ボタン → 手動でも実行可
• 完全自動モードではスケジュールで自動実行される
• 収集結果はキュー一覧タブに表示
✅ ステップ 2: 承認（半自動モードの場合）
半自動モードでは、収集されたツイートをあなたが確認して承認/スキップ。
• キュー一覧タブ → 各ツイートの「承認」or「スキップ」ボタン
• スキップ時は理由を選択可（的外れ、重複、低品質 etc）
• 完全自動モードでは自動承認されるため、このステップは不要
✍️ ステップ 3: AIコメント生成
承認済みツイートに対して、AIが日本語の引用RTコメントを自動生成。
• 8種類のテンプレート（市場インパクト型、速報型、警告型 etc）をAIが自動選択
• 選定設定タブでペルソナ・トーン・NGワード等をカスタマイズ可
• 操作タブの「今すぐ生成」ボタンで手動実行も可
📤 ステップ 4: 引用RT投稿
生成されたコメント付きで引用RTをXに投稿。
• 操作タブの投稿候補一覧から1件選んで即時投稿
• 1回の操作で1件だけ投稿（アカウントBAN防止）
• 投稿済みのツイートは投稿履歴タブで確認可
🎯 カスタマイズ（選定設定タブ）
• 週間フォーカス: 今週特に注目したいテーマを指定
• 注目トピック / 回避トピック: 収集するジャンルの優先度を調整
• ブースト / ブロック アカウント: 特定の海外アカウントを優先or除外
• 引用RTプロンプト設定: ペルソナ名、口調、NGワード、テンプレート選択等
   → AIが生成するコメントの雰囲気を自分好みに調整可
🔗 ツイートURL手動追加
自動収集以外に、引用RTしたいツイートのURLを操作タブから手動で追加可能。
追加後は「AIコメント生成」→「引用RT投稿」の順で実行してください。
⚡ 運用モードの違い
🛡️ 半自動（承認制）: 収集→あなたが承認→生成→投稿。安全だが手間がかかる。
🚀 完全自動: 収集→自動承認→生成→自動投稿。スケジュール通りに全自動で運用。手間ゼロ。

→ 設定タブの「運用モード」で切り替え可能。
